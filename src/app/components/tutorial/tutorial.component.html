Image converter

Hier ist ein sehr nützliches Bash Script. Es wandelt image Dateien in viele nützliche größen für die Webentwicklung um. Es kann sehr leicht angepasst werden. Für die Umwandlung wird das Programm ImageMagick benötigt. Auf den Meisten Linux Systemen Installiert man es einfach mit folgenden Befehl:

sudo apt-get install imagemagick

Der Text wird einfach in eine Text Datei Kopiert. Diese benennt man dann um. Wichtig ist das die Datei mit .sh endet. Also z.B image-converter.sh. Diese Datei führt man dann mit der Konsole aus. Der Befehl dazu lautet bash image-converter.sh. Dann folgt man den Anweisungen des Scriptes indem man zuerst den Namen der vorhandenen image Datei angibt. Danach wird man nach dem gewünschten image Format gefragt. Gibt man png ein bekommt man automatisch 6 neue image Dateien. Die größe der Bilder wurde dafür im Script vorgegeben und kann nach belieben verändert oder erweitert werden.
