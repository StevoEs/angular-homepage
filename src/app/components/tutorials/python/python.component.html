Youtube Downloader

Ein nützliches Tool womit man auf fast allen Seiten Videos in mp4 oder nur die Ton Spur in mp3 Format Downloaden kann. Es basiert auf dem open source Tool Youtube-Dl. Dieses Python Script vereinfacht euch den umgang damit. Es Prüft beim Start, ob alle nötigen Packete installiert sind. Falls nicht wird euch gefragt ob diese heruntergeladen werden sollen.
Dann führt das Script mit Text aufforderungen durch bis zum Download.

Kopiert den Text und speichert es z.B als python-Dl.py ab. Dann startet ihr das Script mit der Konsole mit dem Befehl python python-Dl.py
